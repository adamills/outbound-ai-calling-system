# Outbound AI Calling System

A production‑ready telephony integration combining **Asterisk PBX**, **Telnyx SIP trunking**, and **ElevenLabs voice AI** to route real outbound calls through a conversational agent.

---

## 🚀 Features
- Real‑time voice synthesis and response using ElevenLabs API
- Configurable outbound routes via Asterisk PJSIP
- Secure SIP trunk registration with Telnyx
- JSON‑based call logging and analytics dashboard
- Tested on WSL2 Ubuntu and Termux mobile environments

---

## 🛠 Tech Stack
- Asterisk PBX
- PJSIP
- Telnyx SIP trunk
- ElevenLabs API
- Python / Flask
